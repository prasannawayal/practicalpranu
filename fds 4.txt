6.Write a Python program to create a histogram of the three species of the Iris data. 
import matplotlib.pyplot as plt 
import seaborn as sns 
iris=sns.load_dataset('Iris') 
plt.figure(figsize=(10,6)) 
for species in iris['species'].unique(): 
    species_data=iris[iris['species']==species] 
    sns.histplot(species_data['sepal_length'],label=species,kde=True) 
plt.xlabel('Sepal Length (cm)') 
plt.ylabel('Frequency') 
plt.title('Sepal Length Distribution by Iris Species') 
plt.legend()

3.Write a Python program to create box plots to see how each feature i.e. Sepal Length, Sepal 
Width, Petal Length, Petal Width are distributed across the three species. 

import matplotlib.pyplot as plt 
import seaborn as sns 
iris=sns.load_dataset('Iris') 
plt.figure(figsize=(12,6)) 
for feature in iris.columns[:-1]: 
    sns.boxplot(x='species',y=feature,data=iris,palette='Set1') 
    plt.title(f'Box Plot of {feature} by Species') 
    plt.ylabel(feature)
